

## Быстрый запуск (через Docker)

Нужен установленный [Docker](https://www.docker.com/products/docker-desktop/) (на Windows/macOS — Docker Desktop, на Linux — Docker Engine + Compose).

1. Скачать проект:
   ```bash
   git clone <ссылка-на-репозиторий>
   cd travel_diary
   ```

2. Запустить:
   ```bash
   docker compose up --build
   ```

3. Открыть в браузере: [http://localhost:8000](http://localhost:8000)

При первом запуске автоматически:
- поднимется база PostgreSQL;
- накатятся миграции;
- загрузятся тестовые путешествия с фото;
- создастся пользователь-администратор.

Чтобы остановить — `Ctrl+C` или `docker compose down` (база и фото при этом не удаляются, они хранятся в Docker-томах).

| Логин | Пароль | Права |
|---|---|---|
| `admin` | `admin12345` | суперпользователь, доступ к `/admin` |
| `teterev` | `trip12345` | обычный пользователь (+ доступ к `/admin`) |
| `Алексей` | `trip12345` | обычный пользователь |

Все настройки (пароли, секретный ключ, доступы) лежат в файле `.env` в корне проекта — их можно поменять перед запуском.



## Возможности

- Регистрация и вход
- Добавление, редактирование и удаление путешествий
- Несколько фото к одному путешествию
- Координаты на карте, стоимость, оценки по 4 критериям
- Список всех путешествий и личный список "Мои путешествия"
